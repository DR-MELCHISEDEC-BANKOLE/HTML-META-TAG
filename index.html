HTML <meta> Tag

The <meta> tag serves to define metadata within an HTML document, which essentially refers to information about the data itself.

Usually residing within the <head> element, <meta> tags handle crucial details like character encoding, page description, keywords, document authorship, and viewport settings.

While metadata remains invisible on the webpage, its machine-readable nature proves invaluable. Browsers utilize this data for content display and page reloads, while search engines rely on it for keyword identification, and various web services benefit from its structured information.

Through the <meta> tag, web designers can actively manage the viewportâ€”the visible area of a webpage for users. An example illustrating this control is seen in the "Setting The Viewport" section below.

In an HTML document, metadata is contained within the <head> section. It includes essential information about the webpage's content and presentation.

For instance:

<head>
  <meta charset="UTF-8">
  <meta name="description" content="Free Web tutorials">
  <meta name="keywords" content="HTML, CSS, JavaScript">
  <meta name="author" content="John Doe">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

These <meta> tags define crucial details such as character encoding, a brief description of the content, relevant keywords for search engines, the author's name, and instructions for the viewport to ensure proper display on various devices.

Here's a concise breakdown of the attributes for the <meta> tag in HTML:

  • charset: Specifies the character encoding for the HTML document.
  • content: Defines the value associated with the http-equiv or name attribute.
  • http-equiv: Includes values such as content-security-policy, content-type, default-style, and refresh, providing an HTTP header for the information or value of the content attribute.
  • name: Contains values like application-name, author, description, generator, keywords, and viewport, specifying specific metadata properties for the webpage.

Global Attributes: The <meta> tag also supports the Global Attributes in HTML.

More Examples:

Define keywords for search engines:

<meta name="keywords" content="HTML, CSS, JavaScript">

Define a description of your web page:

<meta name="description" content="Free Web tutorials for HTML and CSS">

Define the author of a page:

<meta name="author" content="Dr Melchisedec Bankole">

Refresh document every 30 seconds:

<meta http-equiv="refresh" content="30">

Setting the Viewport

The viewport is the user's visible area of a web page. It varies with the device - it will be smaller on a mobile phone than on a computer screen.

You should include the following <meta> element in all your web pages:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

This gives the browser instructions on how to control the page's dimensions and scaling.

The width=device-width part sets the width of the page to follow the screen-width of the device (which will vary depending on the device).

The initial-scale=1.0 part sets the initial zoom level when the page is first loaded by the browser.
